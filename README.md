# Deezer Media Player

Flutter app media player using the Deezer API. Bloc pattern to manage the state of the app. Sembast DB to save data locally.

Packages:

-   after_layout
-   bloc
-   cached_network_image
-   cupertino_icons
-   dio
-   equatable
-   flutter_bloc
-   flutter_sound_lite
-   flutter_svg
-   path_provider
-   provider
-   sembast
